Linear search          : O(n)
Binary Search          : O(log2(n))
Selection Sort         : O(n2)
Bubble Sort            : O(n2)
Insertion Sort         : O(n2)
Heap Sort              : O(nlog(n))
Quick Sort             : O(n2)
Merge Sort             : O(nlog(n))
